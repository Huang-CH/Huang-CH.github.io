    socketserver模块与socket模块的关系 #

    socketserver模块是基于socket模块开发的，socket模块是socketserver的底层模块。

    socketserver模块的用途 #

    socketserver模块主要用来处理并发的TCP客户端请求。（如果自己能写出来比socketserver更好的模块，可以用自己的，但在此之前，所有的TCP服务端都应该使用socketserver模块）

    socketserver模块的使用方法（服务端） # 

    import socketserver

# 必须先自定义一个类,并且必须继承socketserver.BaseRequestHandler类
class MySocketServer(socketserver.BaseRequestHandler):
    # 必须定义个handle方法，所有客户端发起连接请求都会先调用handle()方法
    def handle(self):
        conn = self.request    # 与客户端建立连接
        pass    # 客户端连接成功后执行的代码，例如：登录、上传、下载等

# 多线程实例化对象
server = socketserver.ThreadingTCPServer(('127.0.0.1',9000), MySocketServer)
# 使程序一直运行
server.serve_forever()
