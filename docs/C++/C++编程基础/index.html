    C++简介 #

    C++ 是一种静态类型的、编译式的、通用的、大小写敏感的、不规则的编程语言，支持过程化编程、面向对象编程和泛型编程。

    C++ 是 C 的一个超集，事实上，任何合法的 C 程序都是合法的 C++ 程序。

    C++的开发工具 #

    C++常用的开发工具有Microsoft Visual Studio和Jetbrains CLion。

    C++的基础语法 #

    HelloWorld # 

    // 引入iostream库
#include <iostream>
// 调用名称空间std内定义的所有标识符
using namespace std;

int main() {
    // 输出"Hello,World!"，并换行
    cout << "Hello,World!" << endl;
    // 将程序在此处暂停,等待任意输入后,再继续执行后续代码
    system("pause");
    return 0;
}

    注释 #

    单行注释：// 描述信息 多行注释：/* 描述信息 */

    变量 #

    作用：给指定的一段内存空间起名，方便操作这个段内存空间。

    语法：数据类型 变量名称 = 初始值;

    示例：

    #include <iostream>

using namespace std;

int main() {
	  // 定义一个整型变量a，并赋值数字10
    int a = 10;

    cout << "变量a的值是：" << a << endl;

    system("pause");

	  return 0;
}

    常量 #

    作用：用于记录程序中不可更改的数据

    定义常量的两种方式：

    • #define宏常量：#define 常量名 常量值

      通常在文件上方定义，表示一个常量。

    • 使用const修饰的变量：const 数据类型 常量名 = 常量值

      通常在变量定义前加关键字const，修饰该变量为常量，不可修改。

    示例：

    #include <iostream>

using namespace std;

// 定义宏常量
#define day 7

int main() {
  
	  cout << "一周共有：" << day << "天" << endl;
	
    // 使用const修饰的变量
    const int month = 12;
    
    cout << "一年共有：" << month << "个月" << endl;
    
    system("pause");

	  return 0;
}

    标识符的命名规则 #

    • 标识符不能是关键字
    • 标识符只能由字母、数字、下划线组成
    • 第一个字符必须为字母或下划线
    • 标识符中字母区分大小写

    数据类型 #

    整型 #

    作用：表示整数类型的数据。

    C++中对于占用不同内存空间的整形有四种方式：

    数据类型 占用空间 取值范围
    short（短整型） 2 字节 （-2^15 ~ 2^15-1）
    int（整型） 4 字节 （-2^31 ~ 2^31-1）
    long（长整型） Windows为 4 字节，32位Linux为 4 字节，64位Linux为 8 字节 （-2^31 ~ 2^31-1）
    long long（长长整型） 8 字节 （-2^63 ~ 2^63-1）

    实型（浮点型） #

    作用：用于表示小数。

    浮点型变量分为两种：

    • 单精度：float
    • 双精度：double

    单精度和双精度的区别在于表示的有效数字范围不同

    数据类型 占用内存空间 有效数字范围
    float 4 字节 7 位有效数字
    double 8 字节 15 ~ 16 位有效数字

    示例：

    #include <iostream>

using namespace std;

int main() {
	  // 由于编译器看到浮点型的数据会默认为double类型，因此定义float类型的变量时在值后加上“f”可以避免数据类型的转换
	  float f1 = 3.1415926f;
	  double d1 = 3.1415926;
    // 默认情况下，输出的浮点型只能显示6位有效数字，如果需要显示更多的有效数字，需要做额外的配置
	  cout << "f1的值为：" << f1  << endl;
	  cout << "d1的值为：" << d1 << endl;
    
    // 科学计数法在浮点型中的应用
    float f2 = 2e3;  // 2 * 10^3
	  double d2 = 2e-3;  // 2 * 0.1^3
	  cout << "f2的值为：" << f2 << endl;
	  cout << "d2的值为：" << d2 << endl;
	
    system("pause");

	  return 0;
}

    字符型 #

    作用：字符型变量用于表示单个字符的类型。

    语法：char ch = 'a';

    在C和C++中字符型变量只占用1个字节。字符型变量并不是把字符本身放到内存中存储，而是将对应的ASCii编码放入到存储单元。

    注意：在显示字符型变量时，用单引号将字符括起来，不要用双引号；单引号内只能有一个字符，不可以是字符串。

    示例：

    #include <iostream>

using namespace std;

int main() {
	  // 字符型变量的创建
	  char ch = 'a';
	  cout << "字符型变量a的值为：" << ch << endl;

	  // 字符型变量所占的内存大小
	  cout << "字符型变量a所占的内存大小：" << sizeof(ch) << "字节" << endl;

	  // 字符型变量对应的ASCII编码
	  cout << "字符型变量a所对应的ASCII编码为：" << (int)ch <<  endl;

    system("pause");

	  return 0;
}

    字符串型 #

    作用：用于表示一串字符

    C语言风格的字符串：char 变量名称[] = "字符串值"

    示例：

    #include <iostream>

using namespace std;

int main() {
	  char str[] = "Hello,World!";

	  cout << str << endl;

	  system("pause");

	  return(0);
}

    C++语言风格的字符串：string 变量名称 = "字符串值"

    示例：

    #include <iostream>

using namespace std;

int main() {
	  string str = "Hello,World!";

	  cout << str << endl;

	  system("pause");

	  return(0);
}

    布尔型 #

    作用：表示真或假的值

    布尔（bool）类型只有两个值：

    • true（本质是：1）
    • false（本质是：0）

    布尔（bool）类型占用1字节。

    示例：

    #include <iostream>

using namespace std;

int main() {
	bool flag = true;

	cout << flag << endl;  // 1

	flag = false;

	cout << flag << endl;  // 0

	system("pause");

	return(0);
}

    sizeof #

    作用：利用sizeof可以统计数据类型所占内存大小。

    语法：sizeof(数据类型/标量)

    示例：

    #include <iostream>

using namespace std;

int main() {
	  cout << "short类型所占内存空间为：" << sizeof(short) << "字节"  << endl;
	  cout << "int类型所占内存空间为：" << sizeof(int) << "字节"  << endl;
	  cout << "long类型所占内存空间为：" << sizeof(long) << "字节"  << endl;
	  cout << "long long类型所占内存空间为：" << sizeof(long long) << "字节"  << endl;
    
    cout << "float类型所占内存空间为：" << sizeof(float) << "字节"  << endl;
    cout << "double类型所占内存空间为：" << sizeof(double) << "字节"  << endl;
    
    system("pause");

	  return 0;
}

    转义字符 #

    作用：用于表示一些不能显示出来的ASCII字符

    转义字符 含义 ASCII编码值（十进制）
    \a 报警 007
    \b 退格（BS），将当前位置移到前一列 008
    \t 水平制表（HT）（跳到下一个Tab位置） 009
    \n 换行（LF），将当前位置移到下一行开头 010
    \v 垂直制表（VT） 011
    \f 换页（FF），将当前位置移到下一页开头 012
    \r 回车（CR），将当前位置移到本行开头 013
    \" 双引号字符 034
    \' 单引号字符 039
    ? 代表一个问号 063
    \\ 反斜线字符“\” 092

    获取输入数据 #

    作用：从键盘获取输入的数据

    关键字：cin

    语法：cin >> 变量名称

    示例：

    #include <iostream>

using namespace std;

int main() {
	int a = 0;
	cout << "请输入整型变量的值：";
	cin >> a;
	cout << "您输入的整型变量的值为：" << a << endl;

	system("pause");

	return(0);
}
