    C++简介 #

    C++ 是一种静态类型的、编译式的、通用的、大小写敏感的、不规则的编程语言，支持面向过程、面向对象编程和泛型编程。

    C++ 是 C 的一个超集，事实上，任何合法的 C 程序都是合法的 C++ 程序。

    C++的开发工具 #

    C++常用的开发工具有Microsoft Visual Studio和Jetbrains CLion。

    C++的基础语法 #

    HelloWorld # 

    // 引入iostream库
#include <iostream>

// 调用名称空间std内定义的所有标识符
using namespace std;

int main() {
    // 输出"Hello,World!"，并换行
    cout << "Hello,World!" << endl;
    
    // 将程序在此处暂停,等待任意输入后,再继续执行后续代码
    system("pause");
    
    // 程序正常退出
    return 0;
}

    注释 #

    单行注释：// 描述信息 多行注释：/* 描述信息 */

    变量 #

    作用：给指定的一段内存空间起名，方便操作这个段内存空间。

    语法：数据类型 变量名称 = 初始值;

    示例：

    #include <iostream>
using namespace std;

int main() {
	// 定义一个整型变量a，并赋值数字10
    int a = 10;

    cout << "变量a的值是：" << a << endl;

    system("pause");

	return 0;
}

    常量 #

    作用：用于记录程序中不可更改的数据

    定义常量的两种方式：

    • #define宏常量：#define 常量名 常量值

      通常在文件上方定义，表示一个常量。

    • 使用const修饰的变量：const 数据类型 常量名 = 常量值

      通常在变量定义前加关键字const，修饰该变量为常量，不可修改。

    示例：

    #include <iostream>
using namespace std;

// 定义宏常量
#define day 7

int main() {

    cout << "一周共有：" << day << "天" << endl;
	
    // 使用const修饰的变量
    const int month = 12;
    
    cout << "一年共有：" << month << "个月" << endl;
    
    system("pause");

    return 0;
}

    标识符的命名规则 #

    • 标识符不能是关键字
    • 标识符只能由字母、数字、下划线组成
    • 第一个字符必须为字母或下划线
    • 标识符中字母区分大小写

    数据类型 #

    整型 #

    作用：表示整数类型的数据。

    C++中对于占用不同内存空间的整形有四种方式：

    数据类型 占用空间 取值范围
    short（短整型） 2 字节 （-2^15 ~ 2^15-1）
    int（整型） 4 字节 （-2^31 ~ 2^31-1）
    long（长整型） Windows为 4 字节，32位Linux为 4 字节，64位Linux为 8 字节 （-2^31 ~ 2^31-1）
    long long（长长整型） 8 字节 （-2^63 ~ 2^63-1）

    实型（浮点型） #

    作用：用于表示小数。

    浮点型变量分为两种：

    • 单精度：float
    • 双精度：double

    单精度和双精度的区别在于表示的有效数字范围不同

    数据类型 占用内存空间 有效数字范围
    float 4 字节 7 位有效数字
    double 8 字节 15 ~ 16 位有效数字

    示例：

    #include <iostream>
using namespace std;

int main() {
    // 由于编译器看到浮点型的数据会默认为double类型，因此定义float类型的变量时在值后加上“f”可以避免数据类型的转换
    float f1 = 3.1415926f;
    double d1 = 3.1415926;
    // 默认情况下，输出的浮点型只能显示6位有效数字，如果需要显示更多的有效数字，需要做额外的配置
    cout << "f1的值为：" << f1  << endl;
    cout << "d1的值为：" << d1 << endl;
    
    // 科学计数法在浮点型中的应用
    float f2 = 2e3;  // 2 * 10^3
    double d2 = 2e-3;  // 2 * 0.1^3
    cout << "f2的值为：" << f2 << endl;
    cout << "d2的值为：" << d2 << endl;
	
    system("pause");

    return 0;
}

    字符型 #

    作用：字符型变量用于表示单个字符的类型。

    语法：char ch = 'a';

    在C和C++中字符型变量只占用1个字节。字符型变量并不是把字符本身放到内存中存储，而是将对应的ASCii编码放入到存储单元。

    注意：在显示字符型变量时，用单引号将字符括起来，不要用双引号；单引号内只能有一个字符，不可以是字符串。

    示例：

    #include <iostream>
using namespace std;

int main() {
    // 字符型变量的创建
    char ch = 'a';
    cout << "字符型变量a的值为：" << ch << endl;

    // 字符型变量所占的内存大小
    cout << "字符型变量a所占的内存大小：" << sizeof(ch) << "字节" << endl;

    // 字符型变量对应的ASCII编码
    cout << "字符型变量a所对应的ASCII编码为：" << (int)ch <<  endl;

    system("pause");

    return 0;
}

    字符串型 #

    作用：用于表示一串字符

    C语言风格的字符串：char 变量名称[] = "字符串值"

    示例：

    #include <iostream>
using namespace std;

int main() {
    // 定义一个C语言风格的字符串
    char str[] = "Hello,World!";
    
    cout << str << endl;

    system("pause");

    return(0);
}

    C++语言风格的字符串：string 变量名称 = "字符串值"

    示例：

    #include <iostream>
using namespace std;

int main() {
    // 定义一个C++风格的字符串
    string str = "Hello,World!";
    
    cout << str << endl;

    system("pause");

    return(0);
}

    布尔型 #

    作用：表示真或假的值

    布尔（bool）类型只有两个值：

    • true（本质是：1）
    • false（本质是：0）

    布尔（bool）类型占用1字节。

    C/C++遵循非零即真

    示例：

    #include <iostream>
using namespace std;

int main() {
    // 定义一个布尔类型的变量flag
	bool flag = true;

	cout << flag << endl;  // 1

	flag = false;

	cout << flag << endl;  // 0

	system("pause");

	return(0);
}

    sizeof #

    作用：利用sizeof可以统计数据类型所占内存大小。

    语法：sizeof(数据类型/标量)

    示例：

    #include <iostream>
using namespace std;

int main() {
    cout << "short类型所占内存空间为：" << sizeof(short) << "字节"  << endl;
    cout << "int类型所占内存空间为：" << sizeof(int) << "字节"  << endl;
    cout << "long类型所占内存空间为：" << sizeof(long) << "字节"  << endl;
    cout << "long long类型所占内存空间为：" << sizeof(long long) << "字节"  << endl;
    
    cout << "float类型所占内存空间为：" << sizeof(float) << "字节"  << endl;
    cout << "double类型所占内存空间为：" << sizeof(double) << "字节"  << endl;
    
    system("pause");

    return 0;
}

    转义字符 #

    作用：用于表示一些不能显示出来的ASCII字符

    转义字符 含义 ASCII编码值（十进制）
    \a 报警 007
    \b 退格（BS），将当前位置移到前一列 008
    \t 水平制表（HT）（跳到下一个Tab位置） 009
    \n 换行（LF），将当前位置移到下一行开头 010
    \v 垂直制表（VT） 011
    \f 换页（FF），将当前位置移到下一页开头 012
    \r 回车（CR），将当前位置移到本行开头 013
    \" 双引号字符 034
    \' 单引号字符 039
    ? 代表一个问号 063
    \\ 反斜线字符“\” 092

    运算符 #

    作用：用于执行代码的运算。

    运算符的分类

    运算符类型 作用
    算术运算符 用于处理四则运算
    赋值运算符 用于将表达式的值赋值给变量
    比较运算符 用于表达式的比较，并返回一个布尔值（true或false）
    逻辑运算符 用于根据表达式的值返回布尔值（true或false）

    算数运算符 #

    算术运算符包括以下符号

    运算符 术语 示例 结果
    + 正号 +10 10
    - 负号 -10 -10
    + 加 10 + 5 15
    - 减 10 - 5 5
    * 乘 10 * 5 50
    / 除 10 / 5 2
    % 取模（取余） 10 % 3 1
    ++ 前置递增（先自增，再运算或赋值） a = 2; b = ++a; a = 3; b = 3;
    ++ 后置递增（先运算或赋值，再自增） a = 2; b = a++; a = 3; b = 2;
    – 前置递减（先自减，再运算或赋值） a = 2; b = –a; a = 1; b = 1;
    – 后置递减（先运算或赋值，再自减） a = 2; b = a–; a = 1; b = 2;

    赋值运算符 #

    运算符 术语 示例 结果
    = 赋值 a = 2; b = 3; a = 2; b = 3;
    += 加等于 a = 2; a += 3; a = 5;
    -= 减等于 a = 2; a -= 3; a = -1;
    *= 乘等于 a = 2; a *= 3; a = 6;
    /= 除等于 a = 10; a /= 5; a = 2;
    %= 模等于 a = 3; a %= 2; a = 1;

    比较运算符 #

    运算符 术语 示例 结果
    == 等于 2 == 3 0
    != 不等于 2 != 3 1
    < 小于 2 < 3 1
    > 大于 2 > 3 0
    <= 小于等于 2 <= 3 1
    >= 大于等于 2 >= 3 0

    逻辑运算符 #

    运算符 术语 示例 结果
    ! 非 !a 如果a为true，则!a为false
    && 与 a && b 当a和b都为true，则结果为true，否则为false
    || 或 a || b 当a和b中有一个为true，则结果为true；当a和b都为false时，结果为false

    流程控制 #

    C/C++支持最基本的三种程序运行结构：顺序结构、选择结构、循环结构。

    • 顺序结构：程序按顺序执行，不发生跳转。
    • 选择结构：依据条件是否满足，有选择的执行相应功能。
    • 循环结构：依据条件是否满足，循环多次执行某段代码。

    选择结构 #

    if语句 #

    作用：执行满足条件的语句。

    if语句有三种形式：单行格式if语句、多行格式if语句、多条件的if语句。

    if语句可以进行嵌套。

    单行格式if语句：if(条件){ 满足条件执行的语句 }

    注意：if条件后不要加;

    示例：

    #include <iostream>
using namespace std;

int main() {
	// 用户输入一个整数
	int a = 0;
	cout << "请输入一个整数：";
	cin >> a;
	// 打印用户输入的整数
	cout << "您输入的整数为：" << a << endl;
    
	// 判断用户输入的整数大于100时，打印响应的语句
	if (a > 100) {
		cout << "您输入的整数大于100！" << endl;
	}

	system("pause");

	return(0);
}

    多行格式if语句：if ( 条件 ) { 满足条件的执行语句 } else { 不满足条件的执行语句 }

    示例：

    #include <iostream>
using namespace std;

int main() {
	// 用户输入一个整数
	int a = 0;
	cout << "请输入一个整数：";
	cin >> a;
	// 打印用户输入的整数
	cout << "您输入的整数为：" << a << endl;
	
    // 判断用户输入的整数大于或小于等于100时，打印响应的语句；
	if (a > 100) {
		cout << "您输入的整数大于100" << endl;
	}
	else
	{
		cout << "您输入的整数小于等于100" << endl;
	}
    
	system("pause");

	return(0);
}

    多条件的if语句：if ( 条件1 ) { 满足条件1的执行语句 } else if ( 条件2 ){ 满足条件2的执行语句 } …… else { 所有条件都不满足时的执行语句 }

    示例：

    #include <iostream>
using namespace std;

int main() {
	// 用户输入一个整数
	int a = 0;
	cout << "请输入一个整数：";
	cin >> a;
	// 打印用户输入的整数
	cout << "您输入的整数为：" << a << endl;

	if (a > 100) {
		cout << "您输入的整数大于100" << endl;
	}
	else if (a < 100)
	{
		cout << "您输入的整数小于100" << endl;
	}
	else
	{
		cout << "您输入的整数等于100" << endl;
	}
    
	system("pause");

	return(0);
}

    switch语句 #

    作用：执行多条件分支语句。

    语法：

    switch (表达式) 
{
case 结果1:
    执行语句;
    break;
case 结果2:
    执行语句;
    break;
......
default:
    执行语句;
    break;
}

    示例：

    #include <iostream>
using namespace std;

int main() {
	
	// 提示用户操作
	cout << "[1] 用户登录\n[2] 用户注册\n[3] 退出系统\n请输入功能数字：";
	// 接收用户输入
	int input = 0;
	cin >> input;
	cout << "您输入的数字为：" << input << endl;

	// 通过switch语句根据用户的输入进入到相应的功能
	switch (input)
	{
	case 1:
		cout << "正在进入用户登录，请稍后..."  << endl;
		break;
	case 2:
		cout << "正在进入用户注册，请稍后..." << endl;
		break;
	case 3:
		cout << "正在退出系统，请稍后..." << endl;
		break;
	default:
		cout << "输入错误，请重新输入！" << endl;
		break;
	}

	system("pause");

	return(0);
}

    if语句和switch语句的区别：

    • switch语句的结构清晰，并且在执行效率上优于if语句；
    • switch在判断的值或表达式，只能是整型或者字符型，且不可以是一个范围区间。

    三目运算符（三元运算符） #

    作用：通过三元运算符可以实现简单的判断。

    语法：表达式1 ? 表达式2 : 表达式3;

    语法解释：如果表达式1为true，执行表达式2，并返回表达式2的结果；如果表达式2为true，执行表达式3，并返回表达式3的结果。

    在C++中，如果三元运算符返回的是一个变量，那么此变量可以继续赋值。

    示例：

    #include <iostream>
using namespace std;

int main() {
	
	// 定义3个变量a、b、c
	int a = 10;
	int b = 20;
	int c = 30;

	// 通过三元运算符比较a和b，将较大的值赋值给c
	c = a > b ? a : b;

	// 打印结果
	cout << "c的值是：" << c << endl;  // c的值是：20
    
    // 比较a和b的大小，并将100赋值给返回的变量
    a > b ? a : b = 100

    // 打印结果
    cout << "a的值是：" << a << endl;  // a的值是：10
	cout << "b的值是：" << b << endl;  // b的值是：100
    
	system("pause");

	return(0);
}

    循环结构 #

    while循环语句 #

    作用：当满足循环条件时，执行循环语句。

    语法：while ( 循环条件 ) { 循环语句 };

    语法解释：只要是循环条件为true，就一直执行循环语句。

    示例：

    #include <iostream>
using namespace std;

int main() {

	// 循环输出0~10
	int num = 0;
	while (num <= 10)
	{
		cout << num << endl;
		num++;
	}

	system("pause");

	return(0);
}

    在使用while循环语句时要注意避免死循环

    do…while循环语句 #

    作用：先执行一次循环语句，在判断循环条件是否执行循环语句。

    语法：do { 循环语句 } while ( 循环条件 );

    示例：

    #include <iostream>
using namespace std;

int main() {

	int num = 0;

	do {
		cout << num << endl;
		num++;
	} while (0);

	system("pause");

	return(0);
}

    do...while与while的区别在于do...while先要执行一次循环语句，在进行循环条件的判断。

    for循环语句 #

    作用：当满足循环条件时，执行循环语句。

    语法：for ( 初始表达式; 条件表达式; 循环增量 ) { 循环语句; }

    示例：

    #include <iostream>
using namespace std;

int main() {

	// 循环输出0~10
	for (int i = 0; i <= 10; i++)
	{
		cout << i << endl;
	}

	system("pause");

	return(0);
}

    for循环的结构比较清晰，在日常开发中较为常用

    嵌套循环 #

    作用：在循环体中嵌套循环，用于解决一些特定场景的实际问题。

    示例：

    #include <iostream>
using namespace std;

// 打印机99乘法表
int main() {

	// 循环打印行
	for (int i = 1; i < 10; i++)
	{
		// 循环打印列
		for (int j = 1; j <= i; j++)
		{
			cout << j << "x" << i << "=" << i * j << " ";
		}
		cout << endl;
	}

	system("pause");

	return(0);
}

    break语句 #

    作用：用于跳出选择结构和循环结构。

    使用场景：

    • 在switch选择语句中，case执行完毕跳出并终止switch；
    • 在循环语句中跳出当前循环

    示例：

    #include <iostream>
using namespace std;

int main() {

	// 循环打印0～9，当遇到5时，停止打印
	for (int i = 0; i < 10; i++)
	{
		if (i == 5)
		{
			break;
		}
		cout << i << endl;
	}

	system("pause");

	return(0);
}

    continue语句 #

    作用：跳过当前循环，执行下一次循环。

    示例：

    #include <iostream>
using namespace std;

int main() {

	// 循环打印0～9，当遇到5时，跳过当前循环，执行下一次循环
	for (int i = 0; i < 10; i++)
	{
		if (i == 5)
		{
			continue;
		}
		cout << i << endl;
	}

	system("pause");

	return(0);
}

    goto语句 #

    作用：如果存在goto的标记，当代码执行到goto时会直接跳转到标记的位置。

    示例：

    #include <iostream>
using namespace std;

int main() {

	cout << "1" << endl;
	goto flag;
	cout << "2" << endl;
	cout << "3" << endl;

flag:
	cout << "5" << endl;

	system("pause");

	return(0);
}

    由于goto会造成代码结构混乱，非常不利于阅读，因此开发中不建议使用goto语句

    数组 #

    基本概念：所谓数组，就是一个具有相同类型数据元素的集合。

    特点：

    • 一个数组中，每个元素都是相同类型的数据；
    • 数组占用一块连续的内存空间；
    • 可以通过索引取值。

    一维数组 #

    一维数组共有三种定义方式：

    • 数据类型 数组名称[数组长度];
    • 数据类型 数组名称[数组长度] = {值1, 值2, ...};
    • 数据类型 数组名称[ ] = {值1, 值2, ...};

    一维数组名称的用途：

    • 通过sizeof(数组名称)统计数组所占用的内存空间；
    • 通过cout << 数组名称 << endl;获取数组在内存中的首地址（十六进制）。

    冒泡排序：

    是一种比较常用的排序算法，可以对数组内的元素进行排序。原理是通过数组的第一个位置开始依次比较相邻的元素，如果第一个比第二个的数值大，则交换位置，如果第二个比第三个大，则交换它们的位置，以此类推，执行完毕后，数组的最后一个值一定是数组中最大的值，因此在下一轮排序中就可以将最后一个值排除，即每轮排序的次数都会-1，直到数组内的元素不再需要比较。

    示例：

    #include <iostream>
using namespace std;

// 利用冒泡排序实现数组内元素的升序排列
int main() {
	
	// 定义一个无序数组
	int arr[9] = { 4,2,8,0,5,7,1,3,9 };

	// 冒泡排序，总共排序的轮数为元素个数-1
	for (int i = 0; i < 9 - 1; i++)
	{
		// 循环对比的次数等于元素个数减
		for (int j = 0; j < 9 - i - 1; j++)
		{
			// 如果第一个元素比第二个元素大，则交换两个元素
			if (arr[j] > arr[j+1])
			{
				// 定义一个临时变量，用于保存交换位置时的临时元素数据
				int temp = arr[j];
				arr[j] = arr[j + 1];
				arr[j + 1] = temp;
			
			}
		}
	}

	// 数组排序后的结果
	for (int i = 0; i < 9; i++)
	{
		cout << arr[i] << " ";
	}
	cout << endl;

	system("pause");

	return(0);
}

    二维数组 #

    二维数组是在一维数组的基础上增加了一个维度，二维数组本质上是以数组作为数组元素的数组，即“数组的数组”。不严谨的说，二维数组可以理解为一个有横纵坐标的二维表。

    二维数组有4种定义方式：

    • 数据类型 数组名称 [ 行数 ][ 列数 ];
    • 数据类型 数组名称 [ 行数 ][ 列数 ] = { {数据1,数据2} , {数据3,数据4} };
    • 数据类型 数组名称 [ 行数 ][ 列数 ] = { 数据1, 数据2, 数据3, 数据4 };
    • 数据类型 数组名称 [ ][ 列数 ] = { 数据1, 数据2, 数据3, 数据4 };

    以上4种定义方式，第二种更为直观，可以提高代码的可读性

    二维数组的行和列的访问都是从0开始

    示例：

    #include <iostream>
using namespace std;

int main()
{
	
	// 定义一个3行2列的二维数组
	int arr[3][2] = { {1,2},{3,4},{5,6} };

	// 通过嵌套循环将二维数组的所有值遍历出来
	// 外层循环打印行
	for (size_t i = 0; i < 3; i++)
	{
		// 内层循环打印列
		for (size_t j = 0; j < 2; j++)
		{
			cout << arr[i][j];
		}
        cout << endl;
	}

    system("pause");

	return 0;

    二维数组名称的用途：

    • 通过sizeof(数组名称)统计数组所占用的内存空间；
    • 通过cout << 数组名称 << endl;获取数组在内存中的首地址（十六进制）。

    二维数组应用案例：

    案例描述：有三名同学（001、002、003），在一次考试中的成绩如下

    学号 数学 语文 英语
    1 100 100 100
    2 100 90 90
    3 80 70 60

    请分别输出三名同学的总成绩。

    #include <iostream>
using namespace std;

int main()
{
	
	// 定义一个3行2列的二维数组
	int arr[3][4] = { {1,100,100,100},{2,100,90,90},{3,80,70,60} };

	// 通过嵌套循环将二维数组的所有值遍历出来
	// 外层循环打印行
	for (size_t i = 0; i < 3; i++)
	{
		//定义统计分数总和变量sum
		int sum = 0;
		// 内层循环打印列
		for (size_t j = 0; j < 4; j++)
		{
			if ( j != 0 )
			{
				sum += arr[i][j];
			}
		}
		cout << arr[i][0] << "号学生的总分数为：" << sum << endl;
	}

    system("pause");

	return 0;
}

    函数 #

    作用：将一段经常使用的代码封装起来，减少重复代码，增加代码的复用性。

    一个较大的程序，一般分为若干个程序块，每个程序块实现特定的功能。

    函数的定义 #

    函数的定义一般包含5个部分：

    • 返回值的类型

    • 函数名称

    • 参数列表

      定义函数时的参数称之为“形参”，当调用函数时写入的参数称之为“实参”。

    • 函数体语句

    • return 表达式

      如果函数不需要返回值，可以在定义函数时使用void（表示无类型）作为返回值类型，同时return后为空。

    语法：

    返回值类型 函数名称 (参数列表)
{
    函数体语句
    
    return 表达式
}

    示例：

    // 定义一个求和函数，当传入两个数据类型为整形的参数，可以算出这两参数的求和结果
int sum(int num1,int num2)
{
	int sum = num1 + num2;
	return sum;
}

    函数的调用 #

    作用：使用定义好的函数，执行其内部的函数体语句和获取返回值。

    语法：函数名称(参数)

    示例：

    #include <iostream>
using namespace std;

int sum(int num1,int num2)
{
	int sum = num1 + num2;
	return sum;
}

int main()
{

	// 调用sum函数，并通过变量s接收sum函数的返回值
	int s = sum(1,2);
	
	cout << s << endl;

    system("pause");

	return 0;
}

    常见的函数样式 #

    常见的函数样式有：

    • 无参数无返回值
    • 无参数有返回值
    • 有参数无返回值
    • 有参数有返回值

    函数的声明 #

    作用：告诉编译器函数名称以及如何调用函数。函数的实际主体可以单独定义。

    函数可以多次声明，但是只能定义一次函数

    // 声明 max 函数，可以声明多次
int max(int a, int b);
int max(int a, int b);

// 定义 max 函数，只能定义一次
int max(int a,int b)
{
	return a > b ? a : b;
}

    函数的分文件编写 #

    作用：让代码结构更清晰。

    函数分文件编写一般分为4个部分：

    • 创建后缀名为.h的头文件
    • 在缀名为.h头文件中写函数的声明
    • 创建后缀名为.cpp的源文件
    • 在缀名为.cpp源文件中写函数的定义

    示例：

    // 创建 func.h 头文件
#include <iostream>
using namespace std;

// 在头文件中声明函数
void func(int a, int b);

    // 创建 func.cpp 的源文件
#include "func.h"
