    道经 #

    第一章（体道） #

    道，可道也，非恒道也。

    名，可名也，非恒名也。

    无，名萬物之始也；

    有，名萬物之母也。

    故垣无欲也，以觀其眇；

    恒有欲也，以觀其所噭（徼）。

    兩者同出，異名同胃（謂）；

    玄之有（又）玄，衆眇之門。

    简体版：

    道，可道也，非恒道也；

    名，可名也，非恒名也。

    无，名万物之始也；

    有，名万物之母也。

    故恒无欲也，以观其眇；

    恒有欲也，以观其所徼。

    两者同出，异名同谓；

    玄之有玄，众眇之门。

    第二章（养身） #

    天下皆知美之爲美，惡已；

    皆知善，訾不善矣。

    有，无之相生也，

    難，易之相成也，

    長，短之相刑（形）也，

    高，下之相盈也，

    意，聲之相和也，

    先，后之相隋（隨），恒也。

    是以聲（聖）人居无爲之事，行不言之教。

    萬物昔（作）而弗始也，爲而弗志也，成功而弗居也。

    夫唯弗居，是以弗去。

    简体版：

    天下皆知美为美，恶已；

    皆知善，訾不善矣。

    有，无之相生也；

    难，易之相成也；

    长，短之相刑也；

    高，下之相盈也；

    意，声之相和也；

    先，后之相隋，恒也。

    是以圣人居无为之事，行不言之教。

    万物昔而弗始也，为而弗志也，成功而弗居也。

    夫唯弗居，是以弗去。

    第三章（安民） #

    不上賢，使民不争；

    不貴難得之貨，使民不爲盜；

    不見可欲，使民不亂；

    是以聲（聖）人之治也，虛其心，實其腹，弱其志，强其骨。

    恒使民无知无欲也，使夫知不敢，弗爲而已，則无不治矣。

    简体版：

    不上贤，使民不争；

    不贵难得之货，使民不为盗；

    不见可欲，使民不乱；

    是以圣人之治也，虚其心，实其腹，弱其志，强其骨。

    恒使民无知无欲也，使夫知不敢，弗为而已，则无不治矣。

    第四章（无源） #

    道，沖（盅）而用之有弗盈也。

    潚呵，始萬物之宗。

    銼其鋭，解其紛，和其光，同其塵。

    湛呵，似或存。

    吾不知其誰之子也，象帝之先。

    简体版：

    道，盅而用之又弗盈也。

    潚呵！始万物之宗。

    挫其锐，解其纷，和其光，同其尘。

    湛呵！似或存。

    吾不知其谁子也，象帝之先。

    第五章（虚用） #

    天地不仁，以萬物爲芻狗；

    聲（聖）人不仁，以百省爲芻狗。

    天地之間，其猶橐籥與！虛而不淈，蹱而俞出。

    多聞數窮，不若守於中。

    简体版：

    天地不仁，以万物为芻狗。

    圣人不仁，以百省为芻狗。

    天地之间，其猶橐龠舆！虚而不淈，踵而俞出。

    多闻数穷，不若守于中。

    第六章（成象） #

    浴神不死，是胃（謂）玄牝；

    玄牝之門，是胃（謂）天地之根。

    緜緜（綿綿）呵，其若存，用之不堇。

    简体版：

    浴神不死，是谓玄牝；

    玄牝之门，是谓天地之根。

    绵绵呵，其若存，用之不堇。

    第七章（韬光） #

    天長地久。天地之所以能長且久者，以其不自生也，故能長生。

    是以聲（聖）人芮其身而身先，外其身而身存。

    不以其无私輿（與）？故能成其私。

    简体版：

    天长地久。天地之所以能长且久者，以其不自生也，故能长生。

    是以圣人芮其身而身先，外其身而身存。

    不以其无私与？故能成其私。

    第八章（易性） #

    上善治水。水善利萬物而有静，居衆之所惡，故幾於道矣。

    居善地，心善潚，予善天，言善信，正（政）善治，事善能，蹱善時。

    夫唯不静，故无尤。

    简体版：

    上善治水，水善利万物而有静，居众之所恶，故几于道矣。

    居善地，心善潚，予善天，言善信，政善治，事善能，蹱善时。

    夫唯不静，故无尤。

    第九章（运夷） #

    㨁而盈之，不若其已；

    揣而兑之，不可長葆之也；

    金玉盈室，莫之能守也；

    貴富而䮦，自遺咎也；

    功述身芮，天之道也。

    简体版：

    㨁而盈之，不若其已；

    揣而兑之，不可长葆之也；

    金玉盈室，莫之能守也；

    第十章（能为） #

    載營柏（魄）抱一，能毋离（離）乎？

    摶氣至（致）柔，能嬰兒乎？

    脩除玄藍，能毋有疵乎？

    愛民栝（治）國，能毋以知（智）乎？

    天門啓闔，能爲雌乎？

    明白四達，能毋以知乎？

    生之畜之，生而弗有，長而弗宰也，是胃（謂）玄德。

    简体版：

    载营魄抱一，能毋离乎？

    抟气至柔，能婴儿乎？

    脩除玄蓝，能毋疵乎？

    爱民栝国，能毋以知乎？

    天门启阖，能为雌乎？

    明白四达，能毋以知乎？

    生之、畜之，生而弗有，长而弗宰也，是谓玄德。

    第十一章（无用） #

    卅輻同一轂，當其无，有車之用也；

    埏埴而爲器，當其无，有埴器之用也；

    鑿户牖，當其无，有室之用也；

    故有之以爲利，无之以爲用。

    简体版：

    卅辐同一毂，当其无，有车之用也；

    埏埴而为器，当其无，有埴器之用也；

    凿户牖，当其无，有室之用也；

    故有之以为利，无之以为用。

    第十二章（检欲） #

    五色使人目明；

    馳騁田臘使人心發狂；

    難得之貨使人之行方；

    五味使人之口𠷹；

    五音使人之耳聾；

    是以聲（聖）人之治也，爲腹而不爲目，故去罷而耳此。

    简体版：

    五色使人目明；

    驰骋田獵使人心发狂；

    难得之货使人之行妨；

    五味使人之口爽；

    五音使人之耳聋。

    是以圣人之治也，为腹而不为目，故去罢而耳此。

    第十三章（猒恥） #

    龍（寵）辱若驚，貴大梡若身。

    何謂龍辱若驚？龍（寵）之爲下也。得之若驚，失之若驚，是謂龍（寵）辱若驚。

    何謂貴大梡若身？吾所以有大梡者，爲吾有身也；及吾无身，有何梡？

    故貴爲身於爲天下，若可以托天下矣；

    愛以身爲天下，女（如）可以寄天下矣。

    简体版：

    宠辱若惊，贵大梡若身。

    何谓宠辱若惊？宠之为下也，得之若惊，失之若惊，是谓宠辱若惊。

    何为贵大梡若身？吾所以有大梡者，为吾有身也，及吾无身，有何梡？

    故贵为身于为天下，若可以托天下矣。

    爱以身为天下，如可以寄天下矣。

    第十四章（贊玄） #

    視之而弗見，名之曰微；

    聽之而弗聞，名之曰希；

    捪之而弗得，名之曰夷；

    三者不可至計（致詰），故捆而爲一。

    一者，其上不攸，其下不忽，

    尋尋呵！不可名也，復歸於无物。

    是胃（謂）无狀之狀，无物之象，是胃（謂）沕（mì）望。

    隨而不見其後，迎而不見其首。

    執今之道，以御今之有，以知古始，是胃（謂）道紀。

    简体版：

    视之而弗见，名之曰微。

    听之而弗闻，名之曰希。

    捪之而弗得，名之曰夷。

    三者不可至计，故捆而为一。

    一者，其上不攸，其下不忽。

    寻寻呵！不可名也，复归于无物。

    是谓无状之状，无物之象，是谓沕望。

    随而不见其后，迎而不见其首。

    执今之道，以御今之有，以知古始，是谓道纪。

    第十五章（显德） #

    古之善爲道者，微眇（妙）玄達，深不可志。

    夫唯不可志，故强爲之容。

    曰：

    與呵！其若冬涉水；

    猶呵！其若畏四鄰；

    嚴呵！其若客；

    渙呵！其若淩澤；

    沌呵！其若楃；

    湷（混）呵其若濁；

    曠呵！其若浴（谷）；

    濁而情之余清，女（汝）以重之余生。

    葆此道不欲盈，夫唯不欲盈，是以能敝而不成。

    古之善为道者，微眇玄達，深不可志。

    夫唯不可志，故强为之容。

    曰：

    與呵！其若冬涉水。

    猶呵！其若畏四邻。

    严呵！其若客。

    涣呵！其若凌泽。

    沌呵！其若楃。

    湷呵！其若浊。

    旷呵！其若浴。

    浊而情之余清，汝以重之余生。

    葆此道不欲盈，夫唯不欲盈，是以能敝而不成。

    第十六章（归根） #

    至虛，极也，守情，表也。

    萬物旁作，吾以觀其復也。

    天物雲雲，各復歸於其根，曰静。

    静，是谓復命。復命常也，知常明也；不知常，㠵㠵作凶。

    知常容，容乃公，公乃王，王乃天，天乃道，道乃久，沕身不怠。

    简体版：

    至虚，极也，守情，表也。

    万物旁作，吾以观其复也。

    天物云云，各复归于其根，曰静。

    静，是谓复命。复命常也，知常明也；不知常，㠵㠵（huāng）作凶。

    知常容，容乃公，公乃王，王乃天，天乃道，道乃久，沕（mì）身不怠。

    第二十章（异俗） #

    唯與訶，其相去幾何？美與惡，其相去何若？

    人之所畏，亦不可以不畏人。

    朢呵！其未央才。衆人熙熙，若鄉於大牢，而春登臺。

    我泊焉未佻，若嬰儿未咳。

    纍呵！如无所歸。

    衆人皆有餘，我獨遺。

    我禺人之心也，惷惷呵。

    鬻人昭昭，我獨若昏呵。

    鬻人蔡蔡，我獨闷闷呵。

    忽呵，其若海。

    朢呵，其若无所止。

    衆人皆有以，我獨䦎以悝。

    我欲獨異於人，而貴食母。

    简体版：

    唯与诃，其相去几何？美与恶，其相去何若？

    人之所畏，亦不可以不畏人。

    望呵！其未央才。众人熙熙，若飨于大牢，而春登台。我泊焉未佻，若婴儿未咳（hái）。

    累呵，如无所归。众人皆有馀，我独遗。我愚人之心也，蠢蠢呵。

    今舍其慈且勇；

    舍其检且广；

    舍其后且先，则必死矣。

    夫慈，以战则胜，以守则固。

    天将建之，汝以慈垣之。

    第三十三章（配天） #

    善爲士者不武，

    善戰者不怒，

    善勝敵者弗與，

    善用人者爲之下。

    是謂不諍（争）之德，

    是謂用人，是謂肥（配）天，古之極（法则）也。

    简体版：

    善为士者不武，

    善战者不怒，

    善胜敌者弗与，

    善用人者为之下。

    是谓不诤（zhèng）之德，

    是谓用人，是谓肥天，古之极也。

    第三十四章（玄用） #

    用兵有言曰：

    吾不敢爲主而爲客，

    吾不敢進寸而芮（退）尺。

    是謂行无行，襄（乙本作攘）无臂，執无兵，乃无敵矣。

    禍莫大於无適（敵），无適（敵）斤（乙本作近）亡吾葆矣。

    故稱兵（乙本作抗兵，舉兵也）相若（當也），則哀（哀，慈愛）者勝矣（乙本作“而依者朕”）。

    简体版：

    用兵有言曰：

    吾不敢为主而为客，

    吾不敢进寸而芮（ruì）尺。

    是谓行无行，攘无臂，执无兵，乃无敌矣。

    祸莫大于无敌，无敌近亡吾葆矣。

    故称兵相若，则哀者胜矣。

    第三十五章（知難） #

    吾言，甚易知也，甚易行也；

    而人莫之能知也，而莫之能行也。

    言有君，事有宗。

    夫唯无知也，是以不我知。知者希，則我貴矣。（知者希，甲本句残，今本作“知我者希”）

    是以聖人被褐而懷玉。（被褐，謂衣著粗陋，與俗人無別）

    简体版：

    吾言，甚易知也，甚易行也；

    而人莫之能知也，而莫之能行也。

    言有君，事有宗。

    夫唯无知也，是以不我知。知者希，则我贵矣。

    是以圣人被褐而怀玉。

    第三十六章（知病） #

    知不知，尚矣；

    不知不知，病矣。

    是以聖人之不病也，以其病病（動賓結构，懼怕困憂）也，是以不病。

    简体版：

    知不知，尚矣；

    不知不知，病矣。

    是以圣人之不病也，以其病病也，是以不病。

    第三十七章（愛己） #

    民之不畏畏（動賓結构，名詞之“畏”一作“威”），則大畏（威）將至矣。

    毋閘（同狹）其所居，毋猒（厭）其所生。

    夫唯弗猒（厭，同壓），是以不厭（yan4，惡也）。

    是以聖人自知而不自見也，自愛而不自貴也。

    故去被（同彼，乙本作罷）而取此。

    简体版：

    民之不畏畏，则大畏将至矣。

    毋闸其所居，毋猒（yā）其所生。

    夫唯弗猒（yā），是以不厭（yàn）。

    是以圣人自知而不自见也，自爱而不自贵也。

    故去彼而取此。

    第三十八章（任爲） #

    勇於敢者則殺，勇於不敢者則栝（活）。

    此兩者或利或害，天之所亞（惡），孰知其故？

    天之道，

    不單（戰）而善朕（勝），

    不言而善應，不召而自來，

    弾（坦，寬也，安也。乙本作單）而善謀。

    天罔（網）恢恢，疏而不失。

    简体版：

    勇于敢者则杀，勇于不敢者则栝。

    此两者或利或害，天之所恶，孰知其故？

    天之道，

    不战而善胜，

    不言而善应，

    不召而自来，

    弹（dàn）而善谋。

    天网恢恢，疏而不失。

    第三十九章（制惑） #

    若民恒且不畏死，奈何以殺愳（懼，乙本作“䂂”）之也？

    若民恒是死，則而爲者吾將得而殺之，夫孰敢矣。

    若民恒且必畏死（必畏死之畏，犯法獄死之謂也），則恒有司殺者。

    夫代司殺者殺，是代大匠斲（zhuo2）也。

    夫代大匠斲者，則希不傷其手矣。

    人之飢也，以其取食稅之多也，是以飢。

    百姓之不治也，以其上之有以爲也，是以不治。

    民之坙（乙本作輕）死也，以其求生（生聚之生，非生死之生也）之厚也，是以坙（乙本作輕）死。

    夫唯无以生爲者，是賢貴生。

    简体版：

    人之饥也，以其取食税之多也，是以饥。

    百姓之不治也，以其上之有以为也，是以不治。

    民之轻死也，以其求生之厚也，是以轻死。

    夫唯无以生为者，是贤贵生。

    第四十一章（戒强） #

    人之生也柔弱，其死也堅强。

    萬物草木之生也柔脆，其死也枯槁。

    故曰：

    堅强者死之徒也；柔弱微細生之徒也。

    是以兵强則不勝，木强則恒，故强大居下，柔弱微細居上。

    简体版：

    人之生也柔弱，其死也坚强。

    万物草木之生也柔脆，其死也枯槁。

    故曰：

    坚强者死之徒也；柔弱微细生之徒也。

    是以兵强则不胜，木强则恒，故强大居下，柔弱微细居上。

    第四十二章（天道） #

    天之道，酉（猶）張弓者也。

    高者印（甲乙本均作“印”，同抑）之，下者舉之；

    有餘（乙本作余）者損之，不足者補之。

    故天之道，損有餘而益不足。

    人之道則不然，損不足而奉有餘（乙本作“又余”）。

    夫孰能有餘（乙本作“又余”）而有以取奉（取奉，取法也）於天者？

    唯有（乙本作“又”）道者乎。

    是以聖人爲而弗有，成功而弗居也，若此其不欲見賢也。

    简体版：

    天之道，犹张弓者也。

    高者印之，下者举之；

    有余者损之，不足者补之。

    故天之道，损有余而益不足。

    人之道则不然，损不足而奉有余。

    夫孰能有余而有以取奉于天者？

    唯有道者乎？

    是以圣人为而弗有，成功而弗居也。

    若此， 其不欲见（xiàn）贤也。

    第四十三章（任信） #

    天下莫柔弱於水，而攻堅强者莫之能勝也，以其无以易（變易）之也。

    水（甲本句残，通行本作“柔”）之朕（勝）剛也，弱之朕（勝）强也，天下莫弗知也，而莫能行也。

    是故聖人之言云，曰：

    受邦之訽（垢），是謂社稷之主；

    受邦之不祥，是謂天下之王（往也）。

    正言若反。

    简体版：

    天下莫柔弱于水，而攻坚强者莫之能胜也，以其无以易之也。

    水之胜刚也，弱之胜强也，天下莫弗知也，而莫能行也。

    是故圣人之言云，曰：

    受邦之垢，是谓社稷之主；

    受邦之不祥，是谓天下之王。

    正言若反。

    第四十四章（任契） #

    和（乙本作禾）大怨，必有餘怨，焉可以爲善？

    是以聖（乙本“聖”之下有“人執”二字）右介（契，乙本作“左芥”），而不以責於人。

    故有德司介（契），无德司勶（同徹，貢、助、徹之徹也，周之稅法）。

    夫天道无親，恒與善人。

    简体版：

    和大怨，必有余怨，焉可以为善？

    是以圣右介，而不以责于人。

    故有德司介，无德司彻。

    夫天道无亲，恒与善人。
